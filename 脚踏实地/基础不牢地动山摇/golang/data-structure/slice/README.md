# Slice

## 概述
	切片的设计想法是由动态数组概念而来的，为了开发者可以更加方便的使一个数据结构可以自动增加或减少。

## 切片的数据结构

	切片是对数组一个连续片段的的引用，所以切片是一个引用类型。这个切片可以是整个数组，或者是由起始和终止标识的一些项的子集。
切片提供了一个指向数组的动态窗口。

	
	```go
	type slice struct {
		array unsafe.Pointer   // 指向一个数组的指针
		len int	//当前切片的长度
		cap int //当前切片的容量，cap总是大于等len的
	}
	```
